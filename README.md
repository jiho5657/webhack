# Phantom

## 문제 설명

어느 연구소의 실험용 암호 시스템에서 데이터가 유출되었다.
암호문과 함께 정체를 알 수 없는 숫자열이 함께 빠져나왔다.
`challenge/output.txt` 를 분석해 플래그를 복구하라.

알려진 것:

- `N` : 2048비트 정수
- `a0`, `a1`, `a2` : 다항식 계수
- `c = flag^5 + a2 * flag^2 + a1 * flag + a0 mod N`
- `q` : 소수
- `data` : `GF(q)` 위의 원소 64개

플래그 형식: `MJSEC{...}`

## 힌트

data에는 패턴이 있다.
